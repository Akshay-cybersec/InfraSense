# InfraSense
AI-powered smart city system that detects invisible infrastructure failures using on-device sensing, crowd intelligence, and real-time heatmaps.

## Tech Stack

### Web Frontend (Admin Dashboard & Heatmap)
- Next.js (React Framework)
- Google Maps JavaScript API (Heatmap Layer)
- Firebase SDK
- Chart.js / Recharts
- Tailwind CSS

### Mobile Application
- React Native
- Android Accelerometer & Gyroscope APIs
- On-device vibration anomaly detection (Edge AI)

### Backend & Cloud
- Firebase Firestore
- Firebase Cloud Functions (Node.js)
- Grid-based anomaly aggregation & validation

### AI / Machine Learning
- Python
- NumPy, SciPy
- Scikit-learn / TensorFlow (Anomaly Detection)
